# Fullstack App Example

CRUD App with simple product items

## Stack

### Frontend
- ReactJS
- Zustand
- Typescript

### Backend
- NodeJS
- ExpressJS (`./backend`)
- NestJS+TypeScript (`./backend-nestjs`)
- Prisma ORM
- SQLite DB

## You can run the app:

`cd backend` OR `cd backend-nestjs`
`npm i`
`npm start`

`cd frontend`
`npm i`
`npm start`
